Fabric CSS

The home of the CSS implementation of Fabric,

FINN's design system.

Introduction

It's important to understand that Fabric CSS is built on top of an open source project called Tailwind. We built on top of Tailwind in order to take advantage of its enormous set of utility classes that can be used to achieve just about anything. You almost never have to write CSS again.

Getting started

CDN (recommended)

The recommended way to use Fabric is by loading it from our Eik CDN.

Since Eik supports aliasing by major versions you can load Fabric by specifing only the major version you're relying upon. This is great, because it allows us to efficiently utilize the browser's cache as the user navigates FINN, and it ensures you're always using the latest minor.patch version.

In HTML:

<link href="https://assets.finn.no/pkg/@fabric-ds/css/v0/fabric.min.css" rel="stylesheet" />

npm (alternative)

Even though we recommend using Fabric through the Eik CDN, you can also install it as an npm package:

$ npm install @fabric-ds/css

The package.json contains the path to the minified CSS under the style key.

Usage with Podium

If you are developing with Podium , Fabric's CSS should only by included by the layout server , not the individual podlets . This prevents accidentally including several versions of Fabric on the same page.
