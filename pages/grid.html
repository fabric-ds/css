Grid

Use the grid (or Flex where that makes more sense) for any layout related task.

Warning

Note that Fabric grid is a way to use CSS display:grid available through Tailwind. Troika Grid on the other hand was an Abstraction, using whatever styling was available (Using a mix between display:grid, display:flex, and display:table).

Common usage

Example: Grid

Used a lot on FINN, Default display:block behavior(Single column) and DOM order on smaller devices, dividing it into 2/3 and 1/3 from medium devices and up. The responsive prefix md: turns the classes on above the medium breakpoint.

Example: Responsive 2/3, 1/3 Grid

Used a lot on FINN, Default display:block behavior(Single column) and DOM order on smaller devices, dividing it into 2/3 and 1/3 from medium devices and up. The responsive prefix md: turns the classes on above the medium breakpoint.

Example: Results list

Simple grid, could be a result page in card view, could be recommendations. Children are divided into a same size grid, the responsive prefixes (md:, lg:) give them 2 columns on medium, and 3 on large devices.
