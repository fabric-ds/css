Page-container-small

Page-container-small is the primary container for HONK pages. Its main job is to give the standardized responsive behavior, keep the width of the main content, and center/left align it on certain screen sizes to mach banner setups. It is considered the outer boundary for content.

Warning

Note that page-container-small is not fully compatible with panel from Troika. Please migrate to the new Box component in Fabric before using this class, or ensure your Panels still have the appropriate vertical-alignment.

Common usage

 Note that the page-container-small class doesn't have to be applied to a main element. You can adjust this as needed.
