Spacing

Utilities for controlling an element's padding/margin.

References

  • Margin in Tailwind
  • Padding in Tailwind
  • Responsive prefix

Fabric config

The Fabric setup supports the px specific sizes.

Available sizes from tailwind-config :

spacing: { 0: '0', 1: '1px', 2: '2px', 4: '4px', 6: '6px', 8: '8px', 10: '10px', 12: '12px', 14: '14px', 16: '16px', 20: '20px', 24: '24px', 28: '28px', 32: '32px', 40: '40px', 44: '44px', 48: '48px', 56: '56px', 64: '64px', 80: '80px', 96: '96px', 112: '112px', 128: '128px', 144: '144px', }

Common usage

These can be used in a number of ways as described in more detail in the Tailwind documentation . The syntax is exactly the same for margins and paddings , just m instead of p.

Padding examples

Direction Syntax Example Explanation
top, right, bottom, left p{t|r|b|l}-{size} pr-48 Padding right 48px
horizontal, vertical p{x|y}-{size} px-48 Padding horizontally 48px
All sides p-{size} p-48 Padding all sides 48 px
Responsive md:p-48 Padding 48px on medium devices and bigger
